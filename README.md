# Traffic Light Project - Arduino Club

This is a simple Arduino project to control a yellow LED like a traffic light.

## Components
- 1 × Arduino Uno R3
- 1 × 220 Ω Resistor
- 1 × Yellow LED

## How to Use
1. Connect the LED with the resistor to Arduino pin 12 and GND.
2. Upload the code `.ino` to Arduino using Arduino IDE.
3. The LED will turn on and off according to the program.
